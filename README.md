## ğŸš€ Getting Started

### Prerequisites

No build tools or dependencies required! This is a simple HTML/CSS/JavaScript project.

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! The portfolio is ready to view

### Customization

To personalize the portfolio with your information:

1. **Update Personal Information** in `index.html`:
   - Name and title in the hero section
   - About me description
   - Education details (degree, university, dates, achievements)
   - Skills and proficiency levels
   - Project descriptions
   - Contact information (email, phone, location)
   - Social media links

2. **Customize Colors** in `styles.css`:
   - Modify CSS variables in the `:root` section to change the color scheme

3. **Add Your Photo**:
   - Replace the placeholder icon in the hero section with your photo
   - Update the `.image-placeholder` div with an `<img>` tag

4. **Update Projects**:
   - Edit project cards with your actual projects
   - Add project images or keep the gradient placeholders

## ğŸ¨ Customization Tips

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #8B5CF6;
    --secondary-color: #EC4899;
    --accent-color: #F59E0B;
    /* Add your preferred colors */
}
```

### Adding Your Photo

Replace the icon placeholder in `index.html`:

```html
<div class="hero-image">
    <img src="your-photo.jpg" alt="Your Name" class="profile-photo">
</div>
```

### Contact Form Integration

The form currently logs to console. To make it functional:

1. Sign up for a form service like Formspree, EmailJS, or Netlify Forms
2. Update the form action in `index.html` or modify the JavaScript in `script.js`
